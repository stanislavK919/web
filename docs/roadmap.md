# Roadmap ToDo-системи

## Функції на найближчі 3–4 тижні
1. Реалізація повної реєстрації та авторизації користувачів.  
2. CRUD операції для завдань (створення, редагування, видалення).  
3. Фільтрування та сортування завдань за датою і статусом.  
4. Позначення завдань як виконаних та відображення статистики виконання.
