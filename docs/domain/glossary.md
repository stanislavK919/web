## Глосарій
1. Завдання (Task): Окрема одиниця роботи, яку користувач має виконати. Має чіткий Статус.
2. Список (TaskList): Іменована група, що об'єднує логічно пов'язані Завдання.
3. Користувач (User): Зареєстрований обліковий запис, який є власником Списків та Завдань.
4. Автентифікація (Auth): Процес перевірки особи Користувача (зазвичай через логін та пароль) для надання йому доступу до системи.
